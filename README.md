# DataBases

This repository contains practical work related to **relational and NoSQL databases**, including queries, schemas, and database objects implemented using **Microsoft SQL Server**, **PostgreSQL**, and **MongoDB**.

The repository combines isolated query exercises with a semester-long database project based on a real-world business case.

---

## Contents

### SQL and NoSQL practice
The repository includes examples and exercises covering:

- SQL queries (basic to advanced)
- Database schema design
- Constraints and relationships
- Views
- Stored procedures and functions
- Triggers and processes
- Cursors
- Data manipulation and aggregation

These are implemented across:
- **Microsoft SQL Server**
- **PostgreSQL**
- **MongoDB**

Both relational and document-based approaches are explored to compare data modeling and querying strategies.

---

## Semester Project: Coffee Franchise Database (Juan Valdez Case Study)

A central part of this repository is a semester-long project based on the real-world coffee franchise **Juan Valdez**.

### Project overview
- The business model of the franchise was analyzed and emulated
- The menu, shops, and operational structure were replicated
- A complete database system was designed using **fictional data**

### Modeled entities include:
- Shops
- Products and menu items
- Employees
- Clients
- Sales and transactions
- Providers 

All data used is **entirely fictional** and created for educational purposes.
